## 📦 نصب و راه‌اندازی

### الزامات
- Python 3.8+
- pip

### مراحل نصب

```bash
# 1. Clone کردن پروژه
git clone https://github.com/Sepehr1389050/chat-guard-bot.git
cd chat-guard-bot

# 2. نصب وابستگی‌ها
pip install -r requirements.txt

# 3. تنظیم توکن
cp .env.example .env
# سپس توکن را در فایل .env وارد کنید
```

## 🚀 شروع ربات

```bash
python chat_guard_bot.py
```

## 🔑 دریافت توکن تلگرم

1. به [@BotFather](https://t.me/BotFather) در تلگرم پیام بدهید
2. دستور `/newbot` را ارسال کنید
3. نام و username ربات را انتخاب کنید
4. توکن را کپی کنید و در `.env` قرار دهید

## 🔧 Troubleshooting

### مشکل: ربات پاسخ نمی‌دهد

✅ چک کنید:
- توکن درست است؟
- ربات ادمین گروه است؟
- اتصال اینترنت برقرار است؟

### مشکل: خطای دیتابیس

✅ حل:
```bash
rm data.json  # فایل دیتابیس را حذف کنید
python chat_guard_bot.py  # دوباره شروع کنید
```
